# Jam-in-a-Box — IBM waiops-tech-jam Labs (clone de conteúdo)

Cópia completa do conteúdo textual de todas as páginas do site
[https://ibm.github.io/waiops-tech-jam/labs/jam-in-a-box/](https://ibm.github.io/waiops-tech-jam/labs/jam-in-a-box/)
(todas as abas e sub-abas do menu: Jam-in-a-Box, Instana Labs, Turbonomic Labs,
Cloud Pak for AIOps Labs e Concert Labs), obtida diretamente do repositório
fonte [IBM/waiops-tech-jam](https://github.com/IBM/waiops-tech-jam).

- **112 páginas** de conteúdo (arquivos `.mdx`/`.md`) + **24** arquivos de
  configuração de sidebar (`_category_.json`), com estrutura de pastas idêntica
  ao repositório original.
- O texto, código, links e formatação estão exatamente como no original.
- **As imagens não foram baixadas** (são ~1120 arquivos binários — inviável
  baixar em massa neste ambiente). Os links `![...](...)` foram reescritos
  para apontar diretamente para os arquivos originais em
  `raw.githubusercontent.com/IBM/waiops-tech-jam/main/...`, então elas
  continuam aparecendo normalmente ao visualizar os arquivos no GitHub.
